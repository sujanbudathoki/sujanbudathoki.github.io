  • Mediflow Module


    Mediflow Module is role based CMS based on ASP.NET MVC . It is a system which contains 2 roles developer and supporter where developer can insert, update and delete procedures, HTML settings and notify changes to supporters where supporters can insert hospital details and able to only read HTML settings and data.

    CMS Link - http://202.51.75.248:5001/

    More Details

    Framework - ASP.NET MVC
    ORM - Dapper
    Database - SQL Server
    Js Library - Toastr , Alertify
    More Tools - Bootstrap , jquery , jquery-ajax

  • Digital Attendance


    Digital Attendance is a web application based on ASP.NET MVC build for my 7th sem minor project which provides attendance related features based on web. It is a software developed for daily evaluation of students in their continuous assessment record, and performance in accordance with the principle of the institution. It is facilitated to access the performance and information of attendance of a particular Student in a particular semester of study. The information is sorted by the teachers, instructors and advisors, as provided by the student for a particular day throughout a complete semester. This system will also enable the evaluation of student regular presence in various lectures which will determine the eligibility of the student to sit for a semester examination.

    Project Link - https://github.com/sujanbudathoki/Digital-Attendance

    More Details

    Framework - ASP.NET MVC
    ORM - Entity Framework
    Database - SQL Server
    Js Library - Toastr , Alertify
    Front-End Libraries - Materialize css


    "Digital Attendance is an attendance management system based on asp.net mvc and entity framework which i build for 7th sem minor project."
